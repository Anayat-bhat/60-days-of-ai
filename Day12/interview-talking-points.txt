STAR Story 3: Circuit.IQ Concurrency Challenge (One Microsoft — Own the Stack)
Situation: Beta testers reported degraded latency under load. Task: Diagnose and fix without sacrificing response quality. Action: Profiled FastAPI request handling, identified synchronous bottlenecks in the simulation engine, switched to async handlers and added request queuing. STAR Story 4: DistilBERT Fine-Tuning (Technical Rigor)
Situation: Pre-trained sentiment models performed poorly on domain-specific product reviews. Task: Build a classifier that actually works on the target distribution. Action: Curated a balanced training set, implemented stratified train/val/test splits, fine-tuned DistilBERT with a classification head, monitored for overfitting with validation loss curves. Result: 91% accuracy on held-out test set — not train set accuracy, not validation accuracy.
